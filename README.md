# Squads
An alternative Microsoft Teams cllient

> **Note** : Due to API differences between organization and personal accounts, Squads doesn't support personal accounts.
# Run

```
node backend.mjs
```

```
npm run dev
```


## Authentication

Edit the `tokens.json` file in the backend directory

All requests to the Teams API require OAuth 2.0 access tokens. These tokens are numerous and expire quickly. Thankfully, we can generate as many as needed using the Teams token endpoint, which we access using our refresh token (more on that later). However, this rule does not apply to the skypetoken. For the skypetoken, we first need to obtain a skypetoken_asm token from the token endpoint and then make a request to the AuthZ endpoint, which will provide us with the skypetoken.

The refresh token is a unique token that expires in one day. It can be generated by logging in to Teams and is found in local storage. It is automatically obtained using puppeteer.

To sign in change the `email` and `password` values in the `tokens.json` file or use the in-app login page.
