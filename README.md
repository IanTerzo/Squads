# Squads
An alternative Microsoft Teams cllient

## Authentication

All requests to the Teams API require OAuth 2.0 access tokens. These tokens are numerous and expire quickly. Thankfully, we can generate as many as needed using the Teams token endpoint, which we access using our refresh token (more on that later). However, this rule does not apply to the skypetoken. For the skypetoken, we first need to obtain an AuthZ token from the token endpoint and then make a request to the AuthZ endpoint, which will provide us with the skypetoken.

The refresh token is a unique token that expires in one day. It can be generated by logging in to Teams and is found in local storage. I am currently working on a method to renew these tokens to avoid the need to log in every single day. Teams appears to have a system for renewing the refresh tokens without requiring another login. However, for now, you will have to provide your refresh token every time to use Squads.

# Run

```
node backend.mjs
```

